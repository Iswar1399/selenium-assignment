# Selenium Automation Assignment

## Tech Stack
- Java
- Selenium WebDriver
- TestNG
- Rest Assured
- Maven

## Test Cases Covered

### Case 1 - Purchase Flow
- Login using credentials from JSON
- Add items to cart
- Validate cart count
- Complete checkout

### Case 2 - Login Error Validation
- Validate error message
- Footer validation

### Case 3 - Rich Text Editor
- Enter formatted text
- Validate content

### Case 4 - iFrame Handling
- Switch iframe
- Handle new tab
- Form submission

### Case 5 - REST API Testing
- GET /users
- Print name & email
- Validate email format

## How to Run
- Import as Maven project
- Run TestNG tests
